Light version of the BZI Idlers.
- Slimmed down main body, uses 2x M5x20mm instead of M5x30mm
- Heat insert version
- Two color version
- Countersunk version
- Versions for 5mm and 6mm Profiles
- Carrier uses 2x M3x40mm instead of M5x40mm
- Carrier for 20mm and 22mm shafts
